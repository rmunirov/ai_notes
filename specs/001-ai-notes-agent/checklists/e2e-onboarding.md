# E2E онбординг (ручной)

**SC-001**: новый пользователь создаёт первую заметку и задаёт первый вопрос в одном сеансе.

## Чеклист

- [ ] `docker compose up -d` — PostgreSQL
- [ ] `cd backend && uv sync && uv run alembic -c ../alembic.ini upgrade head` (из каталога `backend`, `DB_URL` в `.env`)
- [ ] `cd backend && uv run uvicorn ai_notes.main:app --port 8765`
- [ ] `cd desktop && npm install && npm run dev` — UI на `http://127.0.0.1:5173`
- [ ] Создать заметку (кнопка «Новая заметка», заголовок и текст, авто‑сохранение)
- [ ] В панели агента задать вопрос по содержанию заметки (нужен настроенный `LLM_API_KEY`)
- [ ] Проверить: при пустой базе — подсказка; при отсутствии ключа — 503/сообщение на бэкенде
